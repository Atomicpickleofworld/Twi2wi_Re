# Twi2wi_Re

Lightweight Windows VPN client with AmneziaWG (AWG) and sing-box support.

> ⚠️ Interface is currently in Russian

## 📸 Screenshots

![Main UI](assets/screen1.png)

## 🚀 Features

- ⚡ Fast connection via AmneziaWG (AWG)
- 🔌 sing-box support
- 🧩 Simple and minimal interface
- 🟢 Real-time connection status
- 📦 Installer included
- 🖥️ System tray support

## 📦 Installation

1. Download latest release
2. Run installer
3. Launch Twi2wi_Re
4. Select config → Connect

## ⚙️ Usage

- Choose config
- Click connect
- Done

Status indicators:
- 🔴 Disconnected
- 🟡 Connecting
- 🟢 Connected

## 📁 Supported Configs

### AmneziaWG (AWG)
- .conf files
- Advanced parameters supported

### sing-box
- JSON configs

## ⚠️ Important

- Requires administrator rights
- Do not spam connect button (may cause instability)
- Antivirus may flag VPN tools (false positive)

## 🛠️ Tech Stack

- Python (PyQt6)
- AmneziaWG
- sing-box
- Wintun
